# CasaVista360

Demo inicial de una ficha inmobiliaria inmersiva para presentar propiedades con una experiencia visual mas rica que una galeria tradicional.

## Incluye

- Portada comercial de la propiedad
- Exploracion por planos de planta baja y alta
- Hotspots con detalle por ambiente
- Mensajes de WhatsApp generales y especificos por espacio
- Seccion de entorno y llamada a la accion

## Archivos principales

- `index.html`
- `styles.css`
- `script.js`
- `assets/planta-baja.jpg`
- `assets/planta-alta.jpg`

## Ejecutar localmente

Puedes abrir `index.html` directamente o levantar un servidor simple:

```bash
python3 -m http.server 4173
```
